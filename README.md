# AI-Powered Student Learning Platform

A comprehensive, personalized learning platform for students preparing for Product-based and Service-based companies.

## Features

- **Personalized Learning**: AI-driven roadmaps based on company requirements and student performance
- **Comprehensive Subjects**: OOPS, DBMS, OS, CN, Core CS, Aptitude, DSA, SQL, Programming
- **Practice Engine**: MCQ, Aptitude tests, Topic-wise assessments
- **AI Tutor**: Explanations, summaries, question generation
- **Company-Specific Prep**: 10 Product + 10 Service companies
- **Analytics**: Progress tracking, weakness identification, readiness scores
- **Mock Tests**: Weekly AI-generated tests with PDF reports

## Tech Stack

- **Frontend**: Next.js + Tailwind CSS
- **Backend**: FastAPI + Python
- **Database**: PostgreSQL
- **Authentication**: JWT
- **ORM**: SQLAlchemy
- **AI**: Gemini API
- **Scheduler**: APScheduler
- **PDF Generation**: ReportLab
